Rubric
A rubric is similar to a checklist as it is a list of criteria, but it has levels of quality for each criterion (Brookhart, 2013). They can be useful to evaluate nontechnical skills such as commenting and optimization. Here is an example:
Criteria 2 1 0
Optimization Code executes in <O(logn) =O(logn) >(logn)

Brookhart, S. M. (2013). How to create and use rubrics for formative assessment and grading. Alexandria, VA: ASCD.
